配置 Codex CLI 和 Claude 桌面端连接代理

Codex CLI 配置

在终端中设置环境变量后运行 Codex：

$env:OPENAI_BASE_URL = "http://127.0.0.1:3000/v1" $env:OPENAI_API_KEY = "sk-dummy" codex "你好"

Claude 桌面端配置

编辑 Claude Desktop 配置文件，添加代理设置：

{ "proxy": { "url": "http://127.0.0.1:3000" } }

Windows: %APPDATA%\Claude\claude_desktop_config.json

macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
