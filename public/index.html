Codex CLI 配置

在终端中设置环境变量后运行 Codex：

$env:OPENAI_BASE_URL = "http://127.0.0.1:3000/v1" $env:OPENAI_API_KEY = "sk-dummy" codex "你好"

Claude 桌面端配置

编辑 Claude Desktop 配置文件（claude_desktop_config.json），添加代理设置：

{ "mcpServers": {}, "proxy": { "url": "http://127.0.0.1:3000" } }

配置文件位置：

Windows: %APPDATA%\Claude\claude_desktop_config.json

macOS: ~/Library/Application Support/Claude/claude_desktop_config.json

支持的国产模型

DeepSeek / 通义千问 / 智谱GLM / 月之暗面 / 百川 / 零一万物 / 豆包 / 硅基流动

每个提供商需要单独配置 API Key，可在「提供商」页面选择默认提供商。

OpenAI 兼容直通

也可以直接使用 OpenAI 格式调用任何已配置的提供商：

curl http://127.0.0.1:3000/v1/chat/completions \ -H "Content-Type: application/json" \ -d '{"model":"deepseek-chat","messages":[{"role":"user","content":"Hi"}]}'
